## Requirments

- python => 3.6
- pip => 19
- poetry => 1.2.0

## Instalation

Configure poetry venv inside directory:

```bash
poetry config virtualenvs.in-project true
```

Install, build and publish (locally) packages:

```bash
make setup
```

Now yoy can play the games!
